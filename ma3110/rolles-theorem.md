# Rolle's Theorem

If $f$ is continuous on $[a, b]$ and differentiable on $(a, b)$ and $f(a) = f(b)$, then there exists $c \in (a, b)$ such that
$$f'(c) = 0$$

## Proof
### Case 1: $f$ is a constant function
Then $f'(x) = 0$ for all $x \in (a, b)$.
