# Multinomial Theorem
Generalized [[binomial-theorem]]

$$(x_1 + x_2 + x_3 +... + x_r)^n = \sum_{n_1, n_2, n_3, ..., n_r} \binom{n}{n_1, n_2, ..., n_r} x_1^{n_1} x_2^{n_2} ... x_r^{n_r} $$
